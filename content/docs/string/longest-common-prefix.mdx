```js title="Example"
Input: strs = ["flower","flow","flight"]
Output: "fl"

Input: strs = ["dog","racecar","car"]
Output: ""
```

## Approach

1. **Horizontal Scanning**: Compare the first string with all other strings, finding the common prefix.
2. **Vertical Scanning**: Compare characters at the same position across all strings.
3. **Divide and Conquer**: Split the array into two halves, find LCP of each half, then combine.
4. **Binary Search**: Use binary search on the length of the common prefix.
5. **Trie**: Build a trie and find the longest common prefix path.

## Solution

- Time Complexity: `O(S)`
  > Where S is the sum of all characters in all strings. In the worst case, we need to compare all characters.
- Space Complexity: `O(1)`
  > We only use a constant amount of extra space.

```js tab="JavaScript"
/**
 * @param {string[]} strs
 * @return {string}
 */
var longestCommonPrefix = function(strs) {
    if (strs.length === 0) return "";
    if (strs.length === 1) return strs[0];
    
    // Find the shortest string length
    const minLength = Math.min(...strs.map(str => str.length));
    
    let commonPrefix = "";
    
    // Compare characters at each position
    for (let i = 0; i < minLength; i++) {
        const char = strs[0][i];
        
        // Check if all strings have the same character at position i
        for (let j = 1; j < strs.length; j++) {
            if (strs[j][i] !== char) {
                return commonPrefix;
            }
        }
        
        commonPrefix += char;
    }
    
    return commonPrefix;
};
```

```java tab="Java"
class Solution {
    public String longestCommonPrefix(String[] strs) {
        if (strs.length == 0) return "";
        if (strs.length == 1) return strs[0];
        
        // Find the shortest string length
        int minLength = Integer.MAX_VALUE;
        for (String str : strs) {
            minLength = Math.min(minLength, str.length());
        }
        
        StringBuilder commonPrefix = new StringBuilder();
        
        // Compare characters at each position
        for (int i = 0; i < minLength; i++) {
            char c = strs[0].charAt(i);
            
            // Check if all strings have the same character at position i
            for (int j = 1; j < strs.length; j++) {
                if (strs[j].charAt(i) != c) {
                    return commonPrefix.toString();
                }
            }
            
            commonPrefix.append(c);
        }
        
        return commonPrefix.toString();
    }
}
```
