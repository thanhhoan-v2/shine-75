```js title="Example"
Input: nums = [2, 7, 11, 15], target = 9
Output: [0, 1]
Explanation: Because nums[0] + nums[1] == 9, we return [0, 1].

Input: nums = [3, 2, 4], target = 6
Output: [1, 2]
```

## Approach

1. **Hash Map**: Initialize an empty hash map to store `current_num: index` pairs.
2. **Calculate Complement**: For each element, calculate its complement (target - current_num).
3. **Check Existence**: Check if complement exists in map. If found, return [complement_index, current_index].
4. **Store Current**: If not found, store current_num: current_index in the map.
5. **Single Pass**: Use a single pass through the array for O(n) time complexity.

## Solution

- Time Complexity: `O(n)`
  > Where n is the length of the array. We iterate through the array once.
- Space Complexity: `O(n)`
  > We use a hash map to store the numbers and their indices.

```js tab="JavaScript"
/**
 * @param {number[]} nums
 * @param {number} target
 * @return {number[]}
 */
var twoSum = function(nums, target) {
    const map = new Map();
    
    for (let i = 0; i < nums.length; i++) {
        const complement = target - nums[i];
        
        if (map.has(complement)) {
            return [map.get(complement), i];
        }
        
        map.set(nums[i], i);
    }
    
    return [];
};
```

```java tab="Java"
class Solution {
    public int[] twoSum(int[] nums, int target) {
        Map<Integer, Integer> map = new HashMap<>();
        
        for (int i = 0; i < nums.length; i++) {
            int complement = target - nums[i];
            
            if (map.containsKey(complement)) {
                return new int[] { map.get(complement), i };
            }
            
            map.put(nums[i], i);
        }
        
        return new int[0];
    }
}
```
