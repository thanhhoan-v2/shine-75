```js title="Example"
Input: nums = [-2,1,-3,4,-1,2,1,-5,4]
Output: 6
Explanation: The subarray [4,-1,2,1] has the largest sum 6.
```

## Approach

1. **Kadane's Algorithm**: Use a dynamic programming approach to find the maximum subarray sum
2. **Track Current Sum**: Keep track of the maximum sum ending at the current position
3. **Update Global Maximum**: Compare current sum with global maximum and update if necessary
4. **Reset Current Sum**: If current sum becomes negative, reset it to 0 (start fresh)
5. **Return Result**: Return the global maximum sum

## Solution

- Time Complexity: `O(n)`
  > We iterate through the array once
- Space Complexity: `O(1)`
  > We only use a few variables regardless of input size

```js tab="JavaScript"
/**
 * @param {number[]} nums
 * @return {number}
 */
var maxSubArray = function(nums) {
    let maxSoFar = nums[0];
    let maxEndingHere = nums[0];
    
    for (let i = 1; i < nums.length; i++) {
        maxEndingHere = Math.max(nums[i], maxEndingHere + nums[i]);
        maxSoFar = Math.max(maxSoFar, maxEndingHere);
    }
    
    return maxSoFar;
};
```
