```js title="Example"
Input: nums = [2,3,1,1,4]
Output: true
Explanation: Jump 1 step from index 0 to 1, then 3 steps to the last index.
```

## Approach

1. **Greedy Approach**: Keep track of the maximum reachable position from the current position
2. **Iterate Through Array**: For each position, check if we can reach it from previous positions
3. **Update Maximum Reach**: At each position, update the maximum reachable position
4. **Check Reachability**: If current position is beyond the maximum reachable position, return false
5. **Return Result**: If we can reach the last position, return true

## Solution

- Time Complexity: `O(n)`
  > We iterate through the array once
- Space Complexity: `O(1)`
  > We only use a few variables regardless of input size

```js tab="JavaScript"
/**
 * @param {number[]} nums
 * @return {boolean}
 */
var canJump = function(nums) {
    let maxReach = 0;
    
    for (let i = 0; i < nums.length; i++) {
        if (i > maxReach) {
            return false;
        }
        maxReach = Math.max(maxReach, i + nums[i]);
    }
    
    return true;
};
```
