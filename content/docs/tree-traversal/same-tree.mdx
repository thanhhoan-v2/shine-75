```js title="Example"
Input: p = [1,2,3], q = [1,2,3]
Output: true

Input: p = [1,2], q = [1,null,2]
Output: false

Input: p = [1,2,1], q = [1,1,2]
Output: false
```

## Approach

1. **Recursive DFS**: Use recursive depth-first search to compare trees
2. **Base Cases**: If both nodes are null, return true. If one is null and other isn't, return false
3. **Value Comparison**: Compare the values of current nodes
4. **Recursive Calls**: Recursively check left and right subtrees
5. **Return Result**: Return true only if all comparisons are true

## Solution

- Time Complexity: `O(n)`
  > Where n is the number of nodes in the smaller tree. We visit each node once.
- Space Complexity: `O(h)`
  > The recursion stack depth is at most the height of the smaller tree.

```js tab="JavaScript"
/**
 * @param {TreeNode} p
 * @param {TreeNode} q
 * @return {boolean}
 */
var isSameTree = function(p, q) {
    if (p === null && q === null) return true;
    if (p === null || q === null) return false;
    
    if (p.val !== q.val) return false;
    
    return isSameTree(p.left, q.left) && isSameTree(p.right, q.right);
};
```

```java tab="Java"
class Solution {
    public boolean isSameTree(TreeNode p, TreeNode q) {
        if (p == null && q == null) return true;
        if (p == null || q == null) return false;
        
        if (p.val != q.val) return false;
        
        return isSameTree(p.left, q.left) && isSameTree(p.right, q.right);
    }
}
```
