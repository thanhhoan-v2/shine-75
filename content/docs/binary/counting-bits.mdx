```js title="Example"
Input: n = 2
Output: [0, 1, 1]
Explanation: 0 --> 0 (0 ones), 1 --> 1 (1 one), 2 --> 10 (1 one)

Input: n = 5
Output: [0, 1, 1, 2, 1, 2]
```

## Approach

1. **Dynamic Programming**: Use dynamic programming to build the result array.
2. **Base Case**: ans[0] = 0 (0 has 0 ones).
3. **Recurrence Relation**: For i > 0, ans[i] = ans[i >> 1] + (i & 1).
4. **Bit Manipulation**: i >> 1 removes the least significant bit, i & 1 gets the least significant bit.
5. **Build Array**: Iterate from 1 to n and apply the recurrence relation.

## Solution

- Time Complexity: `O(n)`
  > Where n is the input integer. We iterate through all numbers from 0 to n.
- Space Complexity: `O(n)`
  > We create an array of size n + 1 to store the result.

```js tab="JavaScript"
/**
 * @param {number} n
 * @return {number[]}
 */
var countBits = function(n) {
    const ans = new Array(n + 1).fill(0);
    
    for (let i = 1; i <= n; i++) {
        ans[i] = ans[i >> 1] + (i & 1);
    }
    
    return ans;
};
```

```java tab="Java"
class Solution {
    public int[] countBits(int n) {
        int[] ans = new int[n + 1];
        
        for (int i = 1; i <= n; i++) {
            ans[i] = ans[i >> 1] + (i & 1);
        }
        
        return ans;
    }
}
```
