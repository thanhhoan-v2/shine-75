```js title="Example"
findMaxLength([0, 1]); // 2
findMaxLength([0, 1, 0]); // 2
findMaxLength([0, 1, 0, 1]); // 4
```

## Approach

- Use a hash map to store the first occurrence of each count difference
- Treat 0 as -1 and 1 as 1 to track the balance
- When we encounter the same count difference again, we found a valid subarray
- Keep track of the maximum length found

## Solution

- Time Complexity: `O(n)`
  - We iterate through the array once, so the time complexity is O(n)
- Space Complexity: `O(n)`
  - We use a hash map to store count differences, which can grow up to n elements

```js tab="JavaScript"
const findMaxLength = (nums) => {
  const map = new Map();
  map.set(0, -1); // Initialize with count 0 at index -1
  
  let count = 0;
  let maxLength = 0;
  
  for (let i = 0; i < nums.length; i++) {
    count += nums[i] === 1 ? 1 : -1;
    
    if (map.has(count)) {
      maxLength = Math.max(maxLength, i - map.get(count));
    } else {
      map.set(count, i);
    }
  }
  
  return maxLength;
};
```

```java tab="Java"
public int findMaxLength(int[] nums) {
    Map<Integer, Integer> map = new HashMap<>();
    map.put(0, -1); // Initialize with count 0 at index -1
    
    int count = 0;
    int maxLength = 0;
    
    for (int i = 0; i < nums.length; i++) {
        count += nums[i] == 1 ? 1 : -1;
        
        if (map.containsKey(count)) {
            maxLength = Math.max(maxLength, i - map.get(count));
        } else {
            map.put(count, i);
        }
    }
    
    return maxLength;
}
```
