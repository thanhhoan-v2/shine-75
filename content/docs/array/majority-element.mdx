```js title="Example"
Input: nums = [3, 2, 3]
Output: 3
Explanation: 3 appears 2 times, which is more than ⌊3 / 2⌋ = 1.

Input: nums = [2, 2, 1, 1, 1, 2, 2]
Output: 2
```

## Approach

1. **Boyer-Moore Voting Algorithm**: Use the Boyer-Moore voting algorithm for O(1) space.
2. **Initialize Candidate**: Initialize a candidate and count.
3. **Update Count**: If current element equals candidate, increment count.
4. **Change Candidate**: If current element doesn't equal candidate, decrement count.
5. **Reset on Zero**: If count becomes 0, set current element as new candidate.

## Solution

- Time Complexity: `O(n)`
  > Where n is the length of the array. We iterate through the array once.
- Space Complexity: `O(1)`
  > We only use a constant amount of extra space for variables.

```js tab="JavaScript"
/**
 * @param {number[]} nums
 * @return {number}
 */
var majorityElement = function(nums) {
    let candidate = nums[0];
    let count = 1;
    
    for (let i = 1; i < nums.length; i++) {
        if (count === 0) {
            candidate = nums[i];
            count = 1;
        } else if (nums[i] === candidate) {
            count++;
        } else {
            count--;
        }
    }
    
    return candidate;
};
```

```java tab="Java"
class Solution {
    public int majorityElement(int[] nums) {
        int candidate = nums[0];
        int count = 1;
        
        for (int i = 1; i < nums.length; i++) {
            if (count == 0) {
                candidate = nums[i];
                count = 1;
            } else if (nums[i] == candidate) {
                count++;
            } else {
                count--;
            }
        }
        
        return candidate;
    }
}
```
