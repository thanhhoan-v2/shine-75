```js title="Example"
Input: height = [1, 8, 6, 2, 5, 4, 8, 3, 7]
Output: 49
Explanation: The maximum area is obtained by choosing the height[1] = 8 and height[8] = 7.

Input: height = [1, 1]
Output: 1
```

## Approach

1. **Two Pointers**: Use two pointers starting from both ends of the array.
2. **Calculate Area**: Calculate the area between the two pointers using width Ã— min(height[left], height[right]).
3. **Move Smaller Pointer**: Move the pointer with the smaller height inward.
4. **Track Maximum**: Keep track of the maximum area found.
5. **Continue Until Meeting**: Continue until the pointers meet.

## Solution

- Time Complexity: `O(n)`
  > Where n is the length of the height array. We use two pointers that traverse the array from both ends.
- Space Complexity: `O(1)`
  > We only use a constant amount of extra space for variables.

```js tab="JavaScript"
/**
 * @param {number[]} height
 * @return {number}
 */
var maxArea = function(height) {
    let left = 0;
    let right = height.length - 1;
    let maxArea = 0;
    
    while (left < right) {
        const width = right - left;
        const h = Math.min(height[left], height[right]);
        maxArea = Math.max(maxArea, width * h);
        
        if (height[left] < height[right]) {
            left++;
        } else {
            right--;
        }
    }
    
    return maxArea;
};
```
