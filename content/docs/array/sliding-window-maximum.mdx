```js title="Example"
maxSlidingWindow([1, 3, -1, -3, 5, 3, 6, 7], 3); // [3, 3, 5, 5, 6, 7]
maxSlidingWindow([1], 1); // [1]
maxSlidingWindow([1, -1], 1); // [1, -1]
```

## Approach

- Use a monotonic decreasing queue to track potential maximums
- For each element, remove smaller elements from the back of the queue
- Add the current element to the queue
- Remove elements from the front that are outside the current window
- The front of the queue always contains the maximum for the current window

## Solution

- Time Complexity: `O(n)`
  - Each element is pushed and popped at most once from the queue
- Space Complexity: `O(k)`
  - The queue can store at most k elements

```js tab="JavaScript"
const maxSlidingWindow = (nums, k) => {
  const result = [];
  const queue = []; // Store indices
  
  for (let i = 0; i < nums.length; i++) {
    // Remove elements outside the window
    while (queue.length > 0 && queue[0] <= i - k) {
      queue.shift();
    }
    
    // Remove smaller elements from the back
    while (queue.length > 0 && nums[queue[queue.length - 1]] < nums[i]) {
      queue.pop();
    }
    
    queue.push(i);
    
    // Add maximum to result when window is full
    if (i >= k - 1) {
      result.push(nums[queue[0]]);
    }
  }
  
  return result;
};
```

```java tab="Java"
public int[] maxSlidingWindow(int[] nums, int k) {
    List<Integer> result = new ArrayList<>();
    Deque<Integer> queue = new LinkedList<>(); // Store indices
    
    for (int i = 0; i < nums.length; i++) {
        // Remove elements outside the window
        while (!queue.isEmpty() && queue.peekFirst() <= i - k) {
            queue.pollFirst();
        }
        
        // Remove smaller elements from the back
        while (!queue.isEmpty() && nums[queue.peekLast()] < nums[i]) {
            queue.pollLast();
        }
        
        queue.offerLast(i);
        
        // Add maximum to result when window is full
        if (i >= k - 1) {
            result.add(nums[queue.peekFirst()]);
        }
    }
    
    return result.stream().mapToInt(Integer::intValue).toArray();
}
```
