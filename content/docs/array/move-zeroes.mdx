```js title="Example"
Input: nums = [0, 1, 0, 3, 12]
Output: [1, 3, 12, 0, 0]
Explanation: Move all zeros to the end while maintaining the relative order of non-zero elements.

Input: nums = [0]
Output: [0]
```

## Approach

1. **Two Pointers**: Use two pointers: one for the position to place non-zero elements, another to scan the array.
2. **Swap Non-Zero Elements**: When we find a non-zero element, swap it with the position pointer.
3. **Increment Position**: Increment the position pointer after each swap.
4. **Maintain Order**: All zeros will naturally end up at the end.
5. **In-Place Operation**: Perform the operation in-place without extra space.

## Solution

- Time Complexity: `O(n)`
  > Where n is the length of the array. We iterate through the array once.
- Space Complexity: `O(1)`
  > We only use a constant amount of extra space for variables.

```js tab="JavaScript"
/**
 * @param {number[]} nums
 * @return {void} Do not return anything, modify nums in-place instead.
 */
var moveZeroes = function(nums) {
    let nonZeroIndex = 0;
    
    for (let i = 0; i < nums.length; i++) {
        if (nums[i] !== 0) {
            [nums[nonZeroIndex], nums[i]] = [nums[i], nums[nonZeroIndex]];
            nonZeroIndex++;
        }
    }
};
```

```java tab="Java"
class Solution {
    public void moveZeroes(int[] nums) {
        int nonZeroIndex = 0;
        
        for (int i = 0; i < nums.length; i++) {
            if (nums[i] != 0) {
                int temp = nums[nonZeroIndex];
                nums[nonZeroIndex] = nums[i];
                nums[i] = temp;
                nonZeroIndex++;
            }
        }
    }
}
```
