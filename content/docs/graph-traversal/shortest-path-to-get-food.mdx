```js title="Example"
Input: grid = [["X","X","X","X","X","X"],["X","*","O","O","O","X"],["X","O","O","#","O","X"],["X","X","X","X","X","X"]]
Output: 3
Explanation: The shortest path to get food is shown.
```

## Approach

1. **Breadth-First Search**: Use BFS to find the shortest path from starting position to food
2. **Queue Initialization**: Start from the position marked with '*'
3. **BFS Traversal**: For each position, explore all four directions
4. **Food Detection**: When we encounter '#', we've found food
5. **Distance Tracking**: Track the distance from starting position
6. **Return Result**: Return the minimum distance to reach food

## Solution

- Time Complexity: `O(m*n)`
  > Where m and n are the dimensions of the grid
- Space Complexity: `O(m*n)`
  > We use a queue and visited matrix

```js tab="JavaScript"
/**
 * @param {character[][]} grid
 * @return {number}
 */
var getFood = function(grid) {
    const m = grid.length;
    const n = grid[0].length;
    const queue = [];
    const visited = Array(m).fill().map(() => Array(n).fill(false));
    
    // Find starting position
    for (let i = 0; i < m; i++) {
        for (let j = 0; j < n; j++) {
            if (grid[i][j] === '*') {
                queue.push([i, j, 0]);
                visited[i][j] = true;
                break;
            }
        }
    }
    
    const directions = [[-1, 0], [1, 0], [0, -1], [0, 1]];
    
    while (queue.length > 0) {
        const [i, j, distance] = queue.shift();
        
        if (grid[i][j] === '#') {
            return distance;
        }
        
        for (const [di, dj] of directions) {
            const ni = i + di;
            const nj = j + dj;
            
            if (ni >= 0 && ni < m && nj >= 0 && nj < n && 
                !visited[ni][nj] && grid[ni][nj] !== 'X') {
                visited[ni][nj] = true;
                queue.push([ni, nj, distance + 1]);
            }
        }
    }
    
    return -1;
};
```

```java tab="Java"
class Solution {
    public int getFood(char[][] grid) {
        int m = grid.length;
        int n = grid[0].length;
        Queue<int[]> queue = new LinkedList<>();
        boolean[][] visited = new boolean[m][n];
        
        // Find starting position
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (grid[i][j] == '*') {
                    queue.offer(new int[]{i, j, 0});
                    visited[i][j] = true;
                    break;
                }
            }
        }
        
        int[][] directions = {{-1, 0}, {1, 0}, {0, -1}, {0, 1}};
        
        while (!queue.isEmpty()) {
            int[] curr = queue.poll();
            int i = curr[0], j = curr[1], distance = curr[2];
            
            if (grid[i][j] == '#') {
                return distance;
            }
            
            for (int[] dir : directions) {
                int ni = i + dir[0];
                int nj = j + dir[1];
                
                if (ni >= 0 && ni < m && nj >= 0 && nj < n && 
                    !visited[ni][nj] && grid[ni][nj] != 'X') {
                    visited[ni][nj] = true;
                    queue.offer(new int[]{ni, nj, distance + 1});
                }
            }
        }
        
        return -1;
    }
}
```
